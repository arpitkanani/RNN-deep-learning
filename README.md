# Recurrent Neural Networks (RNN) with PyTorch

This repository contains implementations and experiments with **Recurrent Neural Network (RNN) architectures** using **PyTorch**.  
The goal of this project is to understand how sequential deep learning models work and how they are applied to real-world problems involving sequential or time-dependent data.

---

## 📚 Topics Covered

This repository explores the following RNN architectures and concepts:

- Basic Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)
- Bidirectional RNN
- Sequence modeling fundamentals
- Training and inference using PyTorch

These implementations focus on understanding how different recurrent architectures process sequential data and capture temporal dependencies.

---

## 🧠 Architectures Implemented

### 1️⃣ Basic RNN
A simple recurrent neural network that processes sequential data by maintaining a hidden state across time steps.

Key concepts:
- Hidden state propagation
- Sequence input processing
- Temporal dependency learning

---

### 2️⃣ LSTM (Long Short-Term Memory)

LSTM networks address the **vanishing gradient problem** in traditional RNNs by introducing memory cells and gating mechanisms.

Key components:
- Forget Gate
- Input Gate
- Output Gate
- Cell State

LSTMs are widely used in:
- Natural Language Processing
- Time-series forecasting
- Speech recognition

---

### 3️⃣ GRU (Gated Recurrent Unit)

GRU is a simplified version of LSTM that combines gates to reduce complexity while maintaining strong performance.

Key components:
- Update Gate
- Reset Gate

Advantages:
- Faster training
- Fewer parameters than LSTM

---

### 4️⃣ Bidirectional RNN

Bidirectional RNN processes sequence data **in both forward and backward directions**, allowing the model to capture context from both past and future.

Applications:
- Named Entity Recognition
- Language modeling
- Text classification
